# Wireshark Packet Analysis Project

## Objective
To capture and analyze network traffic using Wireshark.

## Tool Used
Wireshark

## Steps Performed
1. Installed Wireshark and started packet capture
2. Generated traffic by browsing websites
3. Captured network packets
4. Applied filters for DNS and HTTP traffic
5. Analyzed packet details

## Results
- Observed DNS queries for domain name resolution
- Captured HTTP requests between client and server
- Analyzed packet structure and protocol details

## Learning Outcome
- Understood how data flows over a network
- Learned how to inspect packets in real-time
- Gained hands-on experience in traffic analysis

## Conclusion
This project demonstrates basic network traffic analysis using Wireshark, an essential skill in cybersecurity.
