# Hashcat Password Cracking Project

## Objective
To demonstrate password cracking using Hashcat on hashed passwords.

## Tool Used
Hashcat

## Steps Performed
1. Generated MD5 hash of a sample password
2. Created a custom wordlist
3. Used Hashcat to crack the hash
4. Analyzed results

## Results
- Successfully cracked the hashed password
- Demonstrated vulnerability of weak passwords

## Learning Outcome
- Understood password hashing and cracking
- Learned how attackers exploit weak passwords
- Gained hands-on experience with Hashcat

## Conclusion
This project demonstrates how weak passwords can be cracked using basic techniques and highlights the importance of strong password policies.
