# Nmap Network Scan Project

## Objective
To perform network scanning and identify open ports and services on a system.

## Tool Used
Nmap

## Commands Used
nmap -sS 127.0.0.1  
nmap -sV 127.0.0.1  

## Steps Performed
1. Installed Nmap
2. Ran SYN scan on localhost
3. Ran service version detection
4. Collected scan results

## Results
- Identified open ports on the system
- Detected services running on those ports

## Conclusion
This project demonstrates how attackers and security professionals identify open ports and services for security analysis.
